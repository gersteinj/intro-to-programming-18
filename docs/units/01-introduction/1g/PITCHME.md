# Conditional Statements

## if

*if* some condition is true *then* do some action

```python
x = 5

if x % 2 == 0:
    print(f"{x} is even")
```
@[3](Logical test - will evaluate to determine True or False)
@[4](Block of code will run if logical test is evaluated as True)
