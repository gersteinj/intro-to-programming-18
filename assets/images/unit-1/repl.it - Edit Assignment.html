In this exercise, you will explore some of the methods built into Python to work with string

Handling Capitaliation

  • str.capitalize()
  • str.upper()
  • str.lower()
  • str.title()
  • str.swapcase()

All of the above do pretty much what it sounds like. Each one is called on a string, modifies the string as specified, and returns a modified copy. The original string is unaltered. Example:

"Fred is an adorable cat".swapcase() will return "fRED IS AN ADORABLE CAT"
